## Installation and Running 

1. Clone the repository:
```
   git clone https://github.com/KA3RAM/mini-Todo-List.git
```
2. Navigate into the project folder:
```
cd mini-Todo-List
```
3. Install dependencies:
```
npm install
```
4. Run the project:
```
npm start
```


Hope you enjoy the project!
